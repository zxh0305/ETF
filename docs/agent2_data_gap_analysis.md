### 5.1 短期方案：流动性筛选先行

**可立即实现的功能：**

```python
def filter_by_liquidity(data):
    """基于流动性的初步筛选"""
    return [
        etf for etf in data
        if etf['amount'] >= 100_000_000  # 成交额≥1亿
        and etf['liquidity_level'] in ['高', '中']  # 流动性好
    ]
```

- 按成交额排序
- 输出候选池

### 5.2 中期方案：补充PE/PB数据

**数据源选择优先级：**

| 优先级 | 数据源 | 费用 | 数据完整性 | 推荐理由 |

**Tushare Pro 数据获取方案：**

```python

import tushare as ts
pro = ts.pro_api('your_token')

# 获取指数估值数据
df = pro.index_dailybasic(trade_date='20260417', 
    fields='ts_code,pe_ttm,pb,pe_percentile,pb_percentile')

# 获取指数成分股权重
df = pro.index_weight(index_code='000300.SH', trade_date='20260417')
```

### 5.3 长期方案：完整估值体系
